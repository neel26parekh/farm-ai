## 🛠️ Key Developer Commands
- `npm run dev`: Startup the development server.
- `npm run lint`: Check for potential code quality issues.
- `npm run build`: Verify production-ready build.
- `npx prisma generate`: Update the Prisma client after schema changes.
- `npx prisma db push`: Sync schema changes with the database.
